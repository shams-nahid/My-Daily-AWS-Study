## Single Sign On

- Single sign on for
  - Multiple accounts
  - 3rd party Applications
- Integrated with `AWS Organization`
- Supports
  - SAML 2.0 Markup
  - On Premise AD
- Feature
  - Centralized Permission
  - Centralized Auditing `Cloudtrail`
- Alias
  - If an account has alias `my-account`, then sign in URL can be `my-accout.aws----.com`
